# EDR Investigation Playbooks

This repository showcases real-world endpoint detection and response (EDR) investigation scenarios based on practical SOC experience.

##  Key Focus Areas
- Process Tree Analysis  
- Command-line Investigation  
- Threat Validation (True Positive vs False Positive)  
- MITRE ATT&CK Mapping  
- Endpoint + Network Correlation  

##  Tools Used
- CrowdStrike Falcon  
- Cybereason  
- SIEM  
- Palo Alto Firewall  
- DLP & Proxy Logs  

##  Scenarios Included
- Suspicious PowerShell Execution  
- Abnormal Process Tree Behavior  
- Data Exfiltration Investigation  
- Suspicious Login Activity  

##  Objective
To demonstrate practical threat investigation skills and analysis mindset for EDR and Threat Hunting roles.
